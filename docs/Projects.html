Radiant Music App
Java Console App

Radiant Music App is a dynamic Java console application crafted within a week, employing the AGILE methodology for efficient development. Leveraging the power of a SQL database, this application seamlessly integrates admin and user login functionalities. Users can curate their personal music library, adding tracks and effortlessly monitoring their listening progress, whether it's songs yet to be heard, currently playing, or those already enjoyed. With a robust design that includes custom exceptions, the app ensures a smooth user experience by adeptly handling login and music tracking scenarios.
